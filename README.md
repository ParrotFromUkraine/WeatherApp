## ✨ Особливості / Features

- Поточна погода в будь-якому місті світу
- Відображає температуру, відчувається як, вологість, вітер, тиск
- Іконки погоди, що змінюються залежно від умов
- Адаптивний дизайн (гарно виглядає на телефоні та комп'ютері)
- Темна / світла тема (автоматично або вручну)
- Пошук міст українською та англійською
- Використовується відкритий API (OpenWeatherMap)
